## [1.3.0] — 工程化与性能

> 使用 Claude Code 辅助开发；本轮改动走「分支 → PR → CI → 合并」流程。

不改玩法，专注工程质量：性能、架构、部署与仓库呈现。详见 [历史开发记录](./docs/archive/dev-log-2026-07.md)。

### 性能
- **空间网格加速最近敌人查询**：`findNearestEnemy` 的有限半径查询改用每帧构建一次的空间网格（`ENEMY_GRID_CELL_SIZE`），只扫描半径覆盖的格子，替代对 200+ 敌人的全量线性扫描。
- **时间轴阶段每帧缓存**：`getTimelinePhase()` 按 `elapsedSurvivalMs` 记忆化，一帧内数百次调用只计算一次；不稳定速度倍率、传送开关提到循环外。
- **特效对象池**：伤害数字、冲击圈、死亡粒子、闪电线段改为对象池复用，消除战斗高峰期的 GC 抖动。

### 架构
- **模块化重构**：`src/main.js` 从单文件（约 5,700 行）拆分为配置层 `src/config/`（`balance` / `constants` / `upgrades` / `meta`）与场景层 `src/scene/`（`world` / `enemies` / `weapons` / `combat` / `progression` / `timeline` / `effects` / `hud` / `menus` / `systems`）；`main.js` 仅负责组装 `PrototypeScene` 并启动。

### 工程与呈现
- **CI**：新增 GitHub Actions 构建检查工作流（push / PR 触发 `npm run build`）。
- **上线部署**：部署到 Vercel，README 顶部提供在线试玩链接与徽章。
- **许可证与文档**：新增 MIT `LICENSE` 与 SCP 的 CC BY-SA 署名；补充英文 README 镜像；设计文档统一迁入 `docs/`。

## [1.2.0] — 手感与空间重构

> 使用 Claude Code 辅助开发。

大幅改善操作手感与战场空间，这批改动明显改变了游戏体感。

### 新增
- **摄像机跟随 + 大地图**：世界从单屏 960×540 扩大到 1920×1920，摄像机平滑跟随玩家，终于有了放风筝走位的空间。
- **闪避冲刺（空格）**：朝移动方向瞬冲，带 260ms 无敌帧，冷却 2.2 秒，可穿过敌群脱困。武器 HUD 实时显示闪避冷却状态。
- **敌人自碰撞**：敌人之间互相推挤，不再叠成一个点，数量压力变得真实。

### 变更
- 出怪改为在玩家周围、屏幕外一圈生成（原为固定四边），无论移动到何处都从视野外涌入。
- SCP-500 改为在玩家附近 240–420 距离生成，确保大地图中可被找到。
- Boss 从玩家上方入场，出现时保证在画面内；Boss 设为 immovable，不再被自己召唤的小怪推动。
- 所有 UI 覆盖层（武器选择、升级、构筑面板、事件横幅、停电遮罩、结算界面）锚定到屏幕，不随相机漂移。

### 修复
- **升级界面无法点击**：修复覆盖层因坐标系错位导致升级卡片点击命中区偏移、选项点不了的问题。
- 停电光圈做了世界→屏幕坐标换算，光照始终罩在玩家身上。

## [1.1.0] — Bug 修复与体验优化

> 使用 Claude Code 辅助开发。

低风险、见效快的一批改动。

### 新增
- **升级三选一：重抽与跳过**。每局 3 次重抽，或跳过升级换取少量回血。
- **经验宝石磁吸**：宝石进入拾取范围后加速飞向玩家再结算，并伴随拾取微光反馈（原为瞬间消失）。

### 修复
- `B` 键跳关（开发用）此前在正式游戏中始终生效，玩家误触即跳过，现已用 `DEBUG_MODE` 门控。
- 移除武器选择卡片遗留的调试 `console.log`。
- 统一无敌帧时钟到 `elapsedSurvivalMs`，避免升级暂停期间无敌时间被错误消耗。

## [1.0.0] — 首个可玩版本 (MVP)

> 使用 Cursor 辅助开发。

完整的单局幸存者-like 原型。

### 核心内容
- **6 分钟时间轴**：7 个阶段，敌种混合、数值成长、感知干扰逐步升级。
- **双层难度**：时间轴（不可回避）+ 设施事件（电力故障）。
- **三把武器**：基金会勤务手枪（单体）、收容突破器（近战爆发）、特斯拉发射器（链式 AOE），开局三选一。
- **升级系统**：13 项升级定义，按武器过滤，升级三选一。
- **敌人**：3 种普通敌人 + 3 种精英（冲锋 / 闪现 / 分裂）+ SCP-049 Boss（召唤 + 狂暴）。
- **异常拾取**：SCP-500（治疗）、战斗兴奋剂（精英掉落）。
- **表现**：程序化生成的图形与音效（Web Audio），伤害数字、打击闪白、死亡粒子、屏抖等反馈。
- 中文 UI，胜利 / 失败结算界面。
