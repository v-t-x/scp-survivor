# 许可与商业化准备

> **免责声明**：本文是项目政策与风险清单，不构成法律意见。正式商业发布前应重新核对官方条款，并在必要时取得专业法律意见。

## 当前阶段

项目目前仍是原型完善和作品集开发阶段，尚未宣布商业发行。商业化是未来可能性，不是当前已批准的发行计划。

## 可能的商业化方向

- 提供可下载客户端；
- 在独立游戏平台展示或发布；
- 作为个人作品集与求职项目；
- 在玩法和内容成熟后尝试正式销售；
- 将来发展更完整、更原创的商业版本。

## 当前代码许可

仓库当前代码通过根目录 [MIT License](../LICENSE) 发布。已获得该版本代码副本的接收者可以继续按照该许可证文本所授予的权限使用该副本；未来可以调整新版本、新美术、新音频或专有内容的发布策略，但不能把后续策略变更当作对既有副本许可效果的简单撤回。具体法律效果应在商业发布前专业审查。

## SCP 衍生内容注意事项

SCP Foundation 的 [官方许可指南](https://scp-wiki.wikidot.com/licensing-guide) 要求衍生作品处理具体作品/作者署名，并按 CC BY-SA 3.0 处理相同方式共享问题。Creative Commons 的 [CC BY-SA 3.0 官方说明](https://creativecommons.org/licenses/by-sa/3.0/) 说明该许可允许分享和改编，包括商业用途，但必须遵守署名、相同方式共享与不得附加限制等条件。

这意味着“商业用途可能被允许”不等于“可以忽略署名、素材来源或衍生作品许可”。正式发行前必须按实际使用的 SCP 条目、作者、图片、文字与其他素材逐项复核。SCP Foundation 的 [Image Use Policy](https://scp-wiki.wikidot.com/image-use-policy) 也提醒图片可能具有与页面文字不同的许可来源。

## 外部素材准入

以下素材不得进入生产资源：

- 来源不明；
- 无明确许可证；
- 仅允许个人使用；
- 禁止商业使用但无法被替换；
- 要求与项目发布方式冲突；
- 无法满足署名、通知或再分发条件。

UI/Art Agent 在提交外部素材前必须完成来源登记；Project Lead 在合并前审查，商业发布前再次复查。

## 商业发布前检查

- 复核代码、依赖与构建工具许可证；
- 复核每个 SCP 条目、作者与衍生内容署名；
- 复核图片、字体、音频、图标和动画来源；
- 确认分发包中包含必要许可证与署名；
- 检查商店、DRM、附加条款与 share-alike 条件是否冲突；
- 确认隐私、遥测、自动更新、账号和云服务规则；
- 对目标国家/平台取得必要的专业法律与发行意见。
