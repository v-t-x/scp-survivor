# v1.6.0 — 收容区焕新

基金会设施迎来完整的视觉升级。全新的标题界面、武器库、战术 HUD、设施场景、角色与异常敌人动画，让六分钟收容行动第一次拥有统一而清晰的视觉身份。

## 主要更新

- 标题与武器选择改造成统一的基金会设施终端和武器库。
- 新的战术 HUD 集中显示任务、设施、生命、武器、暂停与静音状态。
- 设施场景加入入口、战斗、维护和污染区域，并会随断电、事件和 Boss 阶段改变表现。
- 玩家与七类 R-17 敌人使用正式像素动画，SCP-049 使用正式像素素材。
- 命中、死亡、攻击、精英预警和异常材质获得更清晰的战斗反馈。
- 暂停、升级、失败和胜利界面统一为基金会终端风格。
