# v1.5.0 — SCP-049 压力模型

本 Release 的功能范围以 `daeb3ed605f29a2d97a30458ee12109561dbaf6d` 为边界。v1.5.0 标签指向包含本文的经核验 release-prep 提交；`daeb3ed605f29a2d97a30458ee12109561dbaf6d` 仅用于界定功能范围，并非该标签的目标提交。

## 玩法与规则

- SCP-049 使用 `normal → frenzy → normal → dying` 状态机；状态 deadline 使用 `elapsedSurvivalMs`，暂停时不会推进。
- 狂热阶段停止移动，在固定环上混合召唤防暴镇压单位、闪现潜行者、复制生物体或安保无人机，并开放 1.35 倍弱点承伤窗口。
- 收容突破器对 Boss 的既有增伤与弱点窗口合并计算，并统一限制为 2.0 倍承伤上限。
- 所有非 Boss 敌人（含复制生物体碎片和 Boss 召唤物）每 6–9 秒尝试递归复制；所有活动敌人共享 230 个硬上限，达到上限时复制或召唤安全截断。

## 测试与工程

- 新增并维护 11 个 Node 回归测试，覆盖 Boss 压力模型与递归复制规则。
- CI 在构建前运行 `node --test`，使规则回归先于构建失败暴露。

## 发布边界

- 多平台与 Windows PoC 仍是独立计划和分支，不代表本版本已发布客户端。
- UI/Art 路线、角色动画路线及橘色角色路线均未随本版本发布。
- 在线 Demo 尚未验证与本标签相同；不得声称它已部署 `v1.5.0`。
