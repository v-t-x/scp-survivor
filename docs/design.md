# SCP 幸存者 — 游戏设计文档

> 与当前可玩版本对齐（`src/main.js`）  
> 最后更新：2026-07-03

---

## 1. 游戏类型与核心循环

**类型**：2D 俯视角幸存者-like 单局原型。

**一句话**：基金会安保官被困收容设施，在 **6 分钟时间轴** 内杀怪升级、应对 **电力故障**，撑到终点与 **SCP-049** 决战并取胜。

**核心循环**：

```
选武器（三选一）
  → 移动 + 自动攻击 + 拾取经验
  → 升级三选一
  → 按时间轴应对敌种 / 感知干扰 / 电力故障
  → 4:00 拾取 SCP-500（可选）
  → 6:00 清场 → SCP-049 Boss 战
  → 击败 Boss → 胜利；生命归零 → 失败
```

---

## 2. 胜负条件

| 结果 | 条件 |
|------|------|
| **胜利** | 生存至 6:00，并在 Boss 战中击败 SCP-049 |
| **失败** | 生命值归零（生存阶段或 Boss 阶段均可） |

---

## 3. 6 分钟时间轴

| 时间 | 阶段名称（HUD） | 刷怪 / 敌种 | 数值成长 | 感知干扰 | 其他 |
|------|-----------------|-------------|----------|----------|------|
| **0:00** | 第一阶段：职员感染 | 仅感染职员；间隔 900ms 起 | 基准 | 无 | 开局 |
| **1:00** | 第二阶段：爬行者渗入 | 爬行者权重上升 | 缓慢成长 | 无 | — |
| **2:00** | 第三阶段：无人机部署 | 安保无人机加入 | 继续成长 | **诱饵**（低频） | — |
| **3:00** | 第四阶段：电力故障 | 爬行者权重额外上升 | 继续成长 | 诱饵维持 | **电力故障**（25s，视野变暗） |
| **4:00** | 第五阶段：异常介入 | 刷怪间隔 ×0.88 | 中高 | **敌人短距传送** | **SCP-500** 生成（一次性） |
| **5:00** | 第六阶段：收容加压 | 额外刷怪概率提高 | 接近上限 | **HUD 乱码/抖动** + **子弹偏移** + 屏抖 | 横幅「Boss 即将到来」 |
| **6:00** | 终局：SCP-049 | **停止常规刷怪** | Boss 独立数值 | **全部关闭** | 清场 → Boss 登场 |

敌种权重与刷怪间隔由 `getDifficultyDirectorState()` 驱动；阶段边界为 60s / 120s / 240s。

---

## 4. 双层难度

### 第 1 层 · 时间轴（不可回避）

- 刷怪间隔缩短、敌种混合、敌人生命/伤害成长
- 感知干扰：诱饵（≥2:00）、传送（≥4:00）、HUD 乱码与子弹偏移（≥5:00）
- Boss 战期间感知干扰 **关闭**，保证可读性

### 第 2 层 · 设施事件（唯一变数）

| 项 | 值 |
|----|-----|
| 事件 | **电力故障**（唯一保留） |
| 触发 | 固定 **3:00** |
| 持续 | 25 秒 |
| 效果 | 视野收窄 + 爬行者权重 +0.2 |

**已删除**：收容不稳定条、红色警报、随机设施事件轮询、节点/终端/协议奖励。

## 5. 武器系统

开局 **三选一**，整局仅使用所选武器。

| 武器 | 定位 | 基础机制 |
|------|------|----------|
| **基金会勤务手枪** | 中远程单体 | 自动瞄准最近敌人；支持多弹丸与穿透升级 |
| **基金会收容突破器** | 近距爆发控场 | 4 发弹匣、2s 装填、散布弹丸、击退、压制减速 |
| **特斯拉发射器** | 链式 AOE | 电弧链击多名敌人，伤害逐链衰减 |

### 突破器保留机制

弹匣、装填、散布射击、击退、压制（减速 + 硬直）、精英技能打断。

### 突破器已删除

护甲破碎、防暴盾、最终弹冲击波、击退碰撞连锁伤害、快速装填。

## 6. 升级池（13 项定义，按武器过滤）

仅 **升级三选一**，无协议奖励界面。

### 通用（所有武器）

| key | 名称 | 效果 |
|-----|------|------|
| `damage` | 伤害提升 | 当前武器伤害 +20% |
| `attackSpeed` | 攻击速度 | 当前武器冷却 -15% |
| `moveSpeed` | 移动速度 | 移速 +10% |
| `maxHealth` | 最大生命值 | 上限 +20，并恢复 20 |
| `pickupRadius` | 拾取范围 | 经验拾取范围 +25% |
| `emergencyHeal` | 紧急治疗 | 立即恢复 25 HP |

### 手枪专属

| key | 名称 | 效果 |
|-----|------|------|
| `projectileCount` | 额外弹丸 | +1 弹丸（上限 5） |
| `penetration` | 弹丸穿透 | 穿透 +1 |

### 突破器专属

| key | 名称 | 效果 |
|-----|------|------|
| `breacherKnockback` | 击退强化 | 击退倍率提升 |
| `breacherSuppression` | 压制弹 | 减速 + 硬直延长 |
| `breacherMagazine` | 扩容弹匣 | 弹匣 +1，装填略缩短 |

### 特斯拉专属

| key | 名称 | 效果 |
|-----|------|------|
| `teslaChains` | 额外链击 | 链击目标 +1（上限 8） |
| `teslaCooldown` | 快速放电 | 冷却 -12% |

### 各武器可用池大小

| 武器 | 可选升级数 |
|------|-----------|
| 手枪 | 8 |
| 突破器 | 9（6 通用 + 3 专属） |
| 特斯拉 | 8（6 通用 + 2 专属） |

`getLevelUpChoices()` 只从当前武器 `isAvailable` 为真的条目中随机 3 个。

## 7. 敌人

### 普通敌人（3 种）

| 类型 | 特点 |
|------|------|
| **感染职员** | 基础近战，均衡 |
| **异常爬行者** | 高速低血 |
| **安保无人机** | 远程，保持距离射击 |

### 精英（3 种，约 1:15 后开始出现）

| 类型 | 特点 |
|------|------|
| **防暴镇压单位** | 正面减伤、冲锋 |
| **闪现潜行者** | 周期性传送 + 冲刺 |
| **复制生物体** | 死亡分裂为小碎片 |

精英击杀掉落多颗经验宝石，低概率掉落 **战斗兴奋剂**（+15 HP、6s 移速加成）。

### Boss · SCP-049

| 属性 | 值 |
|------|-----|
| 出现 | 6:00 生存阶段结束，清场后约 2.8s 登场 |
| 生命 | 2500 |
| 移速 | 85 |
| 接触伤害 | 20 |
| 技能 1 | 每 ~11s 召唤 2–3 只弱化感染职员（首召 5s 后） |
| 技能 2 | HP ≤ 50% 时召唤间隔缩短 40% |
| 视觉 | 深绿色方块 + 「SCP-049」标签 |

击败 Boss → `triggerVictory()`。

**Boss 战武器公平性**（2026-07-03 调参）：

| 武器 | Boss 战特殊规则 |
|------|-----------------|
| 手枪 | 无（基准） |
| 突破器 | 射程按 `range`（230）；优先锁定 Boss；对 Boss **×1.5** 伤害 |
| 特斯拉 | 优先锁定 Boss；**全部链击**打在 SCP-049 身上（过载），避免被召唤物分散伤害 |

## 8. SCP-500

| 项 | 值 |
|----|-----|
| 出现 | 固定 **4:00**，全图 **1 次** |
| 效果 | 恢复 **60 HP** |
| 拾取 | 玩家重叠即消耗，播放治疗音效 |

## 9. HUD 布局（5 块）

| # | 位置 | 内容 |
|---|------|------|
| 1 | 左上 | 生命、已用时间、击杀 |
| 2 | 左上偏下 | 等级、经验条 |
| 3 | 左中 | 当前武器状态 |
| 4 | 左下 | 阶段名称 + 下一节点倒计时；Boss 战显示 Boss 生命 % |
| 5 | 顶中横幅 | 事件提示（电力故障 / SCP-500 / Boss 到来等），非常驻 |

**操作提示**：`TAB` 构筑面板 · `M` 静音

---

## 10. MVP 范围

- 单角色、单地图、单局 6 分钟 + Boss
- 3 把武器、13 项升级定义（按武器过滤）
- 3 普通敌人 + 3 精英 + 1 Boss
- 1 个设施事件（电力故障）+ 1 个异常拾取（SCP-500）
- 中文 UI

---

## 11. 明确不做（当前版本）

- 收容节点修复 / 撤离电梯
- 稳定化终端 / 协议奖励
- 红色警报 / 收容不稳定数值条
- 局内解锁其他武器
- 元进度、多地图、多角色、联机
